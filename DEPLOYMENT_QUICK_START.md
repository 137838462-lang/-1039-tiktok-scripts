## ⚡ 3分钟快速部署（推荐）

### 步骤1：创建GitHub仓库

1. 访问 https://github.com/new
2. 输入仓库名：`1039-tiktok-scripts`
3. 选择 **Public**
4. 点击 **Create repository**

### 步骤2：推送代码

在您的电脑/服务器上运行（替换YOUR_USERNAME为您的GitHub用户名）：

```bash
cd /home/ubuntu/tiktok_scripts_web

# 配置git
git remote add origin https://github.com/YOUR_USERNAME/1039-tiktok-scripts.git
git branch -M main
git push -u origin main
```

### 步骤3：在Vercel部署

1. 访问 https://vercel.com
2. 点击 **Sign Up** 或 **Log In**（选择GitHub登录）
3. 授权Vercel访问您的GitHub
4. 点击 **New Project**
5. 选择 `1039-tiktok-scripts` 仓库
6. 点击 **Deploy**

**完成！** 🎉

您将在1-2分钟内获得永久URL：
```
https://1039-tiktok-scripts.vercel.app
```

## 🆘 遇到问题？

### 问题1：推送到GitHub失败

**解决方案：**
- 确保已创建GitHub仓库
- 使用个人访问令牌(PAT)而不是密码
- [创建PAT教程](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token)

### 问题2：Vercel部署失败

**解决方案：**
- 检查GitHub仓库是否为Public
- 查看Vercel的Build Logs获取详细错误信息
- 确保所有文件都已正确推送

### 问题3：网站加载缓慢

**解决方案：**
- 这是一个完全静态网站，Vercel会自动CDN加速
- 如果仍然缓慢，可在Vercel设置中选择离您最近的地区

## 💡 进阶配置（可选）

### 配置自定义域名

如果您有自己的域名（例如：1039-tiktok.com）：

1. 在Vercel项目设置中找到 **Domains**
2. 点击 **Add Domain**
3. 输入您的域名
4. 按照提示配置DNS记录
5. 等待DNS生效（通常24小时）

### 启用环境变量

如果将来需要添加API或其他配置：

1. 在Vercel项目设置中找到 **Environment Variables**
2. 添加您的变量
3. 重新部署
